## 已知问题

### 1. taskService 测试失败
- **原因**: 数据库 Mock 未正确配置
- **影响**: 单元测试无法独立运行
- **解决方案**: 需要在测试前添加 `vi.mock('../src/services/database')` 

### 2. amazonCrawler 集成测试失败
- **原因**: 测试代码中的 Promise 处理问题
- **影响**: 集成测试无法验证爬虫逻辑
- **解决方案**: 修复测试中的异步断言逻辑

### 3. 测试环境问题
- **原因**: 需要真实数据库连接
- **影响**: 部分测试无法在隔离环境中运行
- **解决方案**: 使用 SQLite 内存数据库进行测试
